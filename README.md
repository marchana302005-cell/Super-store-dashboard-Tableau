# Task 2: Data Visualization and Storytelling

## Objective
Create visualizations that convey a compelling story using Tableau to analyze Superstore data and derive actionable business insights.

## Tools Used
- **Tableau Desktop** - For dashboard creation and data visualization
- **Tableau Public** - For publishing and sharing the dashboard
- **GitHub** - For project documentation and version control

## Dataset
`Sample_Superstore.csv` - Sales data from 2014-2018 containing Orders, Sales, Profit, Category, Region, and Segment details.

## Steps to Follow - Mini Guide

1. **Choose right chart for the data**
   - Bar Chart: Sales by Category to compare performance
   - Map Chart: Profit by Region for geographical analysis  
   - Line Chart: Monthly Sales Trend to identify seasonal patterns

2. **Avoid clutter and overuse of colors**
   - Used clean layouts with minimal colors
   - Consistent color scheme across all visualizations
   - Removed unnecessary gridlines and borders

3. **Highlight key takeaways**
   - Technology category leads with 800K+ sales
   - West region shows highest profit of 100K+
   - Clear Q4 sales spike visible in trend line

4. **Add context to each chart**
   - Added descriptive titles and axis labels
   - Included tooltips for detailed data points
   - Dashboard title explains time period: 2014-2018

5. **Focus on business insights, not just visuals**
   - Identified best performing category and region
   - Discovered seasonal sales patterns for inventory planning
   - Highlighted profit vs sales relationship across regions

6. **Create summary slide/storyboard**
   - Combined 3 key charts into single interactive dashboard
   - Dashboard tells complete story: What sells, Where profit is, When sales peak

## Output

**Live Interactive Dashboard:** 
https://public.tableau.com/app/profile/archana.m2696/viz/monthlysalestrend_17828236691330/Dashboard1

**Dashboard Components:**
1. **Sales by Category** - Bar chart showing Technology, Office Supplies, Furniture performance
2. **Profit by Region** - Geographical map highlighting West region dominance
3. **Monthly Sales Trend** - Time series line chart revealing seasonal Q4 peaks

## Key Business Insights

1. **Product Strategy:** Technology category generates highest revenue - 800K+. Focus marketing budget here.
2. **Regional Focus:** West region delivers maximum profit - 100K+. Replicate success strategies in other regions.
3. **Seasonal Planning:** Sales consistently peak in Q4 Nov-Dec every year. Plan inventory and staffing accordingly.

## Outcome
Successfully mastered visual storytelling by creating an interactive dashboard that transforms raw sales data into clear business insights for decision making.
